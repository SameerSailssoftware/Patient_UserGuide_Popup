Patient Information

This document describes what information to set by default to patient settings, which need to be added in Add New patient or Edit Patient windows. Once the details are filled in, as per the below instructions, then click on the “SAVE” button or enter keyboard shortcut “Alt+S” to save the details.

Instructions to Enter Default Patient Information

1. Select “Patient Insurance Type” as either Cash or any other insurances to which patient is applicable for, from the Dropdown menu.

2. Select “Patient Language” from the dropdown menu.

3. Select “Patient Residence” type from the dropdown menu.

4. Select by default “NO KNOWN ALLERGIES” in “Patient Allergies” field. The user can select the required information from the list while entering the Patient information.

5. Patient History View: Select from dropdown to set a default value like, what span of Patient’s history should be viewed, like All, or Last 1 year, or Last 2 years.

6. Mandate Diagnosis Selection:
a. If the user wants to make the diagnosis selection mandatory in patient window, then make the toggle switch “Yes”

a. If the user doesn’t want to make the diagnosis selection mandatory in patient window, then make the toggle switch “No”

7. Mandate Allergy Selection:
a. If the user wants to make the Allergy selection mandatory in patient window, then make the toggle switch“Yes”

b. If the user doesn’t want to make the Allergy selection mandatory in patient window, then make the toggle switch “No”

8. Send Patient # as Patient ID: Patient number is a unique code which is taken for individual patient in backend, Patient Id is the identity number of the Patient. If user selects “Yes” then Patient number will be taken as Patient Id, if not the user needs to enter Patient ID number.
